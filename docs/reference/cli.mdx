---
title: CLI Reference
---

CLI entrypoint: `ingest-relay`.

## Commands

### `init-db`

Initialize runtime database tables.

```bash
ingest-relay init-db
```

### `run`

Execute one connector run.

```bash
ingest-relay run --connector /absolute/path/to/connector.yaml
```

Options:

- `--connector` (required)
- `--push-run-id` (optional, process existing queued push run)

### `serve`

Start API + Ops UI + Studio UI.

```bash
ingest-relay serve --host 0.0.0.0 --port 8080
```

Options:

- `--host` (default `0.0.0.0`)
- `--port` (default `8080`)
