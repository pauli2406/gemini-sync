IngestRelay uses an AI-agent-first workflow defined in `AGENTS.md`.

## Contribution Baseline

- Follow Red -> Green -> Refactor for behavior changes.
- Update mapped docs per `docs/doc_sync_map.yaml`.
- Keep connector examples policy intact.

## Required PR Evidence

- Failing test/eval evidence before implementation
- Passing test/eval evidence after implementation
- Docs changes and rationale
- Risk tier and gate outcomes
